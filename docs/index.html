Vimursor

Think in Vim. Navigate in Mac.

Download DMG
brew install --cask vimursor

Features

Hint Mode

Cmd+Shift+Space

Click any UI element with keyboard hints. Labels appear on every clickable element — type the label to click instantly.

Demo coming soon

Search Mode

Cmd+Shift+/

Type to search and click matching elements. Incrementally filters UI elements by text, then press Enter to click.

Demo coming soon

Scroll Mode

Cmd+Shift+J

Vim-style scrolling with j/k for line scroll and u/d for half-page scroll. Press Escape to exit.

Demo coming soon

Cursor Mode

Cmd+Shift+K

Move the cursor with hjkl keys. Precise, keyboard-driven cursor positioning without touching the mouse.

Demo coming soon

Keyboard Shortcuts

Shortcut Mode Description
Cmd+Shift+Space Hint Mode Show clickable element labels
Cmd+Shift+/ Search Mode Search and click UI elements by text
Cmd+Shift+J Scroll Mode Vim-style scrolling (j/k/u/d)
Cmd+Shift+K Cursor Mode Move cursor with hjkl
Escape All Modes Exit current mode

brew install --cask vimursor
